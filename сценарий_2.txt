── 15. Ари ──
Для этого есть встроенная Windows-функция — DPAPI, Data Protection API.
Никаких сторонних библиотек, только стандартный ctypes.

Работает просто: берём наши ключи, шифруем через CryptProtectData и пишем бинарный файл keys.enc.
При расшифровке Windows проверяет, что это тот же пользователь на том же компьютере.
Скопировать файл куда-то ещё — бесполезно, без твоей учётки это просто набор байт.

.env убираем. При старте программы keystore читает keys.enc в память,
и дальше всё приложение получает ключи через keystore.get().

  код:
    # Windows DPAPI — шифрование ключей
    #
    # Сохранение:
    # json.dumps(keys) → CryptProtectData → keys.enc
    #
    # Загрузка при старте:
    # keys.enc → CryptUnprotectData → json.loads → _session{}
    #
    # Доступ в коде:
    # keystore.get("GROK_API_KEY")
    #
    # Привязка: Windows user + компьютер
    # Скопировать keys.enc на другой ПК — бесполезно
